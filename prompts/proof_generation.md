## Instructions

### Proof Construction

1. **Select the Best Approach**
   - Review all hypotheses and their evaluations
   - Choose the most promising approach(es)
   - Consider combining multiple approaches if beneficial

2. **Build the Proof Structure**
   - Start with clear definitions and notation
   - State all assumptions explicitly
   - Outline the logical flow before detailing

3. **Write Each Step Carefully**
   - Justify every claim
   - Reference known theorems by name
   - Handle all cases exhaustively
   - Mark any steps that need further verification

4. **Self-Correct**
   - Review your proof for gaps
   - Check all logical transitions
   - Verify edge cases
